# gsmteixeira.github.io

Repositório que hospeda o site pessoal/portfólio acadêmico de **Gabriel Teixeira** em GitHub Pages.

O projeto é uma landing page com foco em Astrofísica, Cosmologia e aplicações de Deep Learning em dados astronômicos. A página apresenta informações gerais, pesquisa, publicações, palestras, projetos e um formulário de contato.

---

## Stack

- **Framework:** React  
- **Linguagem:** TypeScript  
- **Build / Dev server:** Vite  
- **Estilos:** Tailwind CSS v3  
- **UI helpers:** Chakra UI (apenas para containers/layout; estilização principal via Tailwind)  
- **Formulário de contato:** integração com Formspree  

---

## Seções do site

A navegação é feita em uma única página, com uma *navbar* fixa que faz scroll suave para cada seção:

- `#inicio` – Begin  
- `#sobre` – About  
- `#pesquisa` – Research  
- `#publicacoes` – Publications  
- `#palestras` – Talks  
- `#projetos` – Projects  
- `#contato` – Contact
